\chapter{Methodology}
\label{chap:method}

This chapter defines a fake end-to-end pipeline used only for template testing.

\section{Pipeline Overview}
The pipeline contains preprocessing, model fitting, and post-analysis stages. Each stage is represented with deterministic pseudo-settings to keep results reproducible in narrative form \cite{owens2025survey}.

\section{Synthetic Data Construction}
A mock dataset of 10,000 samples is described with balanced classes and controlled noise factors.

\section{Evaluation Metrics}
Primary metrics are accuracy, RMSE, and latency. Secondary metrics include calibration and robustness under perturbation.
